# 📚 Online Learning Platform

A web-based online learning platform that allows users to access courses, watch videos, and manage learning content.

## 🚀 Features

- User login functionality  
- Dashboard for course navigation  
- Video-based learning modules  
- Backend server for handling requests  

## 🛠 Tech Stack

- JavaScript  
- Node.js  
- Express.js  
- HTML, CSS  

## 📂 Project Structure

- App.js – Main frontend logic  
- Login.js – User authentication  
- Dashboard.js – User dashboard  
- Video.js – Video content handling  
- server.js – Backend server  

## ⚙️ How to Run Locally

```bash
git clone https://github.com/Akshaya-Veerappan/ONLINE-LEARNING-PLATFORM.git
cd ONLINE-LEARNING-PLATFORM
npm install
node server.js
